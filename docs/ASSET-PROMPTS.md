## Configuración general para todos los prompts

- **Fondo**: Transparente (PNG) o negro sólido (#06060e)
- **Paleta**: Púrpura (#7c5cbf, #9d6ee7, #c084fc), blanco, negro profundo
- **Estilo**: Minimalista, flat con sutil profundidad 3D, sin bordes gruesos
- **Resolución**: 512x512px mínimo para iconos, 1920x1080px para banners

## Tips para mejores resultados

1. **Leonardo AI**: Usa el modelo "DreamShaper" para iconos flat, "Absolute Reality" para mockups realistas
2. **Ideogram**: Mejor para texto + imagen combinados (útil si quieres el nombre "Vitely" integrado)
3. **Microsoft Designer**: Bueno para fondos y gradientes abstractos
4. **Canva**: Útil para combinar elementos generados y ajustar dimensiones

## Formato de archivo recomendado

| Uso | Formato | Dimensión |
|-----|---------|-----------|
| Iconos | PNG (transparente) | 512x512 |
| Fondos | JPG o WebP | 1920x1080 |
| Logo | SVG o PNG (transparente) | 512x512 |
| Mockups | PNG (transparente) | 800x1200 |

## Ubicación en el proyecto

Los assets generados van en:
```
frontend/src/assets/
├── icons/          ← iconos y logo
├── images/         ← ilustraciones y mockups
└── backgrounds/    ← texturas y fondos (crear si no existe)
```
